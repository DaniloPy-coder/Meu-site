Projeto 6 - Hamburgueria

Clique aqui para acessar o projeto

O site apresenta seções para diferentes tipos de produtos, como hambúrgueres, porções e bebidas, com imagens, descrições e botões para adicionar itens ao carrinho de compras. O cabeçalho exibe o nome e logotipo do restaurante, com uma navegação para facilitar o acesso às seções de produtos. Um botão de carrinho no topo permite que o cliente veja o número de itens adicionados e, ao clicar, abre um modal para o usuário revisar os itens, inserir o endereço e telefone, e finalizar o pedido.
