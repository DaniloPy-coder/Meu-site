Meus Projetos

Conversor de moedas

Esse código implementa um conversor de moedas usando React, com algumas funcionalidades adicionais, como troca de temas (modo claro e escuro) e formatação de grandes números.

Previsão do tempo

O sistema serve para buscar a previsão do tempo por cidade, exibindo a temperatura atual, as condições climáticas (ensolarado, nublado, etc.) e a previsão para os próximos dias. A interface é responsiva, adaptada tanto para dispositivos móveis quanto para desktops.

Big Burger

Desenvolvi o projeto "Big Burger", um site interativo para uma lanchonete fictícia, utilizando HTML, CSS e JavaScript. O objetivo foi criar uma plataforma intuitiva e visualmente atraente para os usuários escolherem seus próprios itens e navegarem pelo cardápio de maneira fluida.

Buscador de cep

O buscador de CEP em React é uma aplicação que permite ao usuário consultar informações de endereço a partir de um CEP inserido. Integrado a uma API como ViaCEP, o sistema valida a entrada, exibe os dados retornados, como rua, bairro e cidade, ou apresenta mensagens de erro para CEPs inválidos.
